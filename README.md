A lightweight Chrome extension designed to help users monitor their browsing habits and improve focus throughout the day.

Features
Time Tracking: Automatically tracks time spent on different websites.

Dashboard: View a summary of your daily browsing activity.

Productivity Insights: Identify which sites consume the most of your time.

Simple Interface: Clean and easy-to-use popup UI.

Installation
Clone the repository to your local machine:

Bash
git clone https://github.com/PragatiNigam29/Smart-Productivity-Tracker-Extension.git
Open Google Chrome and navigate to chrome://extensions/.

Enable Developer mode in the top right corner.

Click the Load unpacked button.

Select the folder where you cloned this repository.

How to Use
Click on the extension icon in your browser toolbar to open the dashboard.

The extension will automatically start tracking your activity on active tabs.

Check the popup to see your current productivity breakdown.

Tech Stack
JavaScript: Logic for tracking and data management.

HTML/CSS: Used for building the popup interface.

Chrome Extension APIs: Utilizes chrome.tabs and chrome.storage to track and save user data.
